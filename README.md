**RPG Panel – Projet RPG en Java Swing**

Ce projet est un mini-jeu RPG développé en Java avec une interface graphique Swing.
Il permet au joueur de :
Créer son personnage (nom + classe)
Voir son inventaire
Acheter des armes dans une boutique
Explorer une carte générée aléatoirement
Affronter des monstres
Détruire des obstacles
Atteindre la sortie pour terminer la partie
